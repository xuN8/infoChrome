My Coding Journey: The Last Five Years

Codecademy Javascript

The first language I learned was Javascript. I had no idea what it was or what it was used for, but I loved it. Practiced on and off since I didn't know how to apply it outside of Codecademy.

Arduino C++

CodingWithKids summer camp at AWS Elemental. Learned about hardware components and how to program Arduino boards. Made a robot with remote control, self driving, and line tracking functionality.

Code.org

Introduced to block coding at school. Code.org has little puzzle problems for learning coding fundamentals, but at this point it was mostly review.

Scratch

Another block-based app from school, but this one is more of an open sandbox. Made some pretty crazy projects on there outside of class.

MIT App Inventor

Probably my least favorite block coding app, but it was still fun to use. Everything was asynchronus and event-based, which felt a little limiting sometimes.

Codecademy HTML/CSS/JS

Finally tried my hand at web development. I found HTML very easy to learn, and made a simple website within a few days of learning. This was around the time ChromeOS Linux came out, so I used VScode for this.

Codecademy Python 2

Had to learn Python eventually. It was so different from Javascript I was gawking at how clean the syntax was.

Discord Bot Python

Because why not? Finally got a proper PC after developing on a Chromebook for three years, you bet I was ready(Heroku wasn't downloading properly on Chrome Linux). Asynchronus programming. Modules. HTTP requests. Web scraping. *cough* *cough* API request limits. Boy that was a lot. At least my bot can get weather, news, and nutrition facts now...
